# 问答

## 为什么我的plugins文件夹没有nodejs文件夹？

Liteloader的版本需要大于2.5.0。

## 如何安装llplugin文件？

直接放进BDS的plugins文件夹即可。

## 出现客户端版本过低？

使用扫码登陆，如果无法扫码，换为go-cqhttp登录模式。

## 如何扫码登录？

将配置文件的`pwd`选项写为`null`
``` json
"pwd":null
```

## 可以连接多个服务器吗？

暂时不可以。

## 如何提交滑块验证码？

输入`spark slider <ticket>`
